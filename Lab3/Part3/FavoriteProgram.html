Project Details
Utilize a touchscreen Raspberry Pi 4b with a Linux based OS to mimic a generic bluetooth keyboard!

I had originally explored utilizing a USB connection to treat a device a virtual keyboard. Security Protocols within USB prevented this approach from fitting the requirements for the device. In my search, I discovered that Bluetooth connectivity utilized specific technologies that apply to my desire to create a more connected universe.

Bluetooth Profiles allow you to set device Hexadecimal Address, Generic Bluetooth name, and a descriptive UUID.

Bluetooth Profiles Hexadecimal Bluetooth Addresses Universally Unique Identifier 
        
# Create a bluetooth device to emulate a HID keyboard/mouse,
# Configure using Hex Address and UUID:
class BluetoothDevice():
  # change the software profile to match a generic keyboard
  MY_ADDRESS = "B8:27:EB:B6:8C:21"
  UUID = "00001124-0000-1000-8000-00805f9b34fb"
  MY_DEV_NAME = "3rd_Bluetooth_Keyboard/Mouse"

The code above sets the Hexadecimal address of a Bluetooth Device to a generic Keyboard, matching that generic figure with a generic UUId with keyboard and mouse privileges.
